Introduction
Bienvenue dans le projet de site web de location de voitures! Ce projet utilise HTML5 et CSS3 pour créer une plateforme en ligne permettant aux utilisateurs de découvrir et de choisir parmi une variété de voitures de luxe, de voitures haut de gamme et de voitures normales. Le site offre une navigation conviviale et une présentation visuelle attrayante pour offrir une expérience utilisateur immersive.

Structure du Projet
Le projet est organisé en utilisant une structure HTML bien définie, avec des sections claires pour l'en-tête, la navigation, les sections principales, la vidéo promotionnelle et le pied de page. Les fichiers sont répartis comme suit :

HTML Files :

index.html : Page d'accueil
VoitureLuxe.html : Page détaillée pour les voitures de luxe
hotGame.html : Page détaillée pour les voitures haut de gamme
MoyenGame.html : Page détaillée pour les voitures normales
contact.html : Page de contact
CSS Files :

styles.css : Feuille de style principale pour le projet.
Images et Vidéos :

Le dossier HTML/IMAGES contient toutes les images utilisées dans le projet.
La vidéo promotionnelle se trouve dans le dossier HTML/IMAGES.
Styles CSS
La feuille de style styles.css définit l'apparence visuelle du site. Voici quelques points saillants de la conception :

Utilisation de Flexbox : La disposition des sections principales est gérée à l'aide de Flexbox pour une présentation flexible et réactive.
Media Queries : Des requêtes média sont utilisées pour rendre le site réactif, offrant une expérience utilisateur optimale sur différents appareils.
Style de la Galerie de Photos : Les images de voitures sont présentées dans des boîtes stylisées avec des marges et des bordures appropriées.
Navigation
La barre de navigation propose des liens vers différentes sections du site, permettant aux utilisateurs de naviguer facilement entre les pages.

Pages Détaillées
Chaque section principale sur la page d'accueil offre un bouton "About" qui redirige l'utilisateur vers une page détaillée spécifique pour en savoir plus sur les catégories de voitures.

Contact
La page de contact propose un formulaire pour permettre aux utilisateurs de soumettre des requêtes et de communiquer avec l'entreprise. Les coordonnées sont également fournies pour faciliter la communication.

Vidéo Promotionnelle
Une vidéo promotionnelle est intégrée à la page d'accueil pour offrir une expérience visuelle dynamique aux visiteurs.

Réseaux Sociaux
Le pied de page propose des liens vers les réseaux sociaux, permettant aux utilisateurs de suivre l'entreprise sur Facebook, Twitter et Instagram.
